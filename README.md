# xhshow-go

基于 Go 语言实现的 xhshow 项目版本  
本项目在原 xhshow Python 项目的基础上进行重构与功能迁移，**约 60% 的代码由 AI 辅助完成转化与优化**，其余部分为人工重写与结构调整。

---

## 项目说明

`xhshow-go` 是一个使用 **Go（Golang）** 实现的 xhshow 版本，目标是：

- 提升执行性能与并发能力
- 更适合服务端 / 长时间运行场景
- 为后续桌面端（Wails）或 API 服务化做准备

---

##  AI 参与说明

> 本项目中约 **60% 的代码** 由 AI 辅助完成，包括但不限于：

- Python → Go 的逻辑转化
- 函数结构与模块拆分建议
- 注释与代码规范整理

其余核心逻辑、项目结构设计、功能取舍与最终调试均由人工完成。
